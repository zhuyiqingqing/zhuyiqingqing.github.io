    # 一、HTML初体验

    # 1、安装sublime软件

    # 2、创建项目文件夹及文件

    # 2.1 创建项目文件夹，命名“个人简历”，并在其中创建子文件夹，命名为images或img,用于存放图片。

    # 2.2 将项目文件夹拖入sublime中

    # 2.3 按【Ctrl+N】创建文件后，按【Ctrl+S】保存，并命名为“index.html”.

    # 2.4 .输入html5按回车，

    # 2.5 输入h1-h6添加标题，输入p添加段落文字，hr水平线
